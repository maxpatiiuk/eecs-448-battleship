defineView

Defines a view from passed in name and gets the view template from the associated component dir.

defineView(view: any)

renderView

Renders a defined view into a container. Passes in neccessary, predefined render parameters.

renderView(viewName: any, container: any, renderOptions: any)
Parameters
viewName (any) View name
container (any) Container to render the view within
renderOptions (any) Array of view-specific rendering options
Throws
  • any: Undefined if view is undefined for this context
